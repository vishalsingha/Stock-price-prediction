# Stock-price-prediction of  APPLE Inc.

This is the closing Stock Price Prediction of Apple Inc. from 2012 to 2020 taken by Yahoo.com . In this project 1600 working days (2012-01-03  to  2018-05-11) closing stock data is used to predict the closing stock prices of 311 days from (2018-10-05 to 2019-12-31) .

## Requirements
1. [numpy](https://numpy.org/)
2. [pandas](https://pypi.org/project/pandas/)
3. [matplotlib](https://matplotlib.org/)
4. [sklearn](https://scikit-learn.org/stable/)
5. [tensorflow](https://www.tensorflow.org/)
6. [keras](https://keras.io/)


## Model Used
[Long Short Term Memory(LSTM)](https://colah.github.io/posts/2015-08-Understanding-LSTMs/)
